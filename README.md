🎮 무법자 칼치기 - 게임 설명

게임 이름: 무법자 칼치기

장르: 레이싱 / 회피 게임

게임 소개:
도로 위에서 내려오는 자동차들을 피하며 살아남는 스릴 넘치는 레이싱 게임입니다.
플레이어는 자신의 자동차를 좌우로 움직여 장애물(다른 자동차)을 피하면서 최대한 오래 살아남아 점수를 쌓아야 합니다.

게임 특징:

스마트 스크롤 배경: 도로가 아래로 스크롤되어 실제 운전하는 느낌을 제공합니다.

플레이어 자동차 조작: 키보드 좌우 방향키를 사용하여 자동차 이동 가능

장애물 자동차 3종류: 위에서 내려오는 장애물 자동차가 랜덤하게 나타납니다.

점수 시스템: 장애물을 피하면서 시간당 점수가 쌓입니다. 오래 생존할수록 높은 점수 획득

게임 종료: 장애물과 충돌 시 게임 종료, 최종 점수를 화면에 표시

조작 방법:

← : 왼쪽으로 이동

→ : 오른쪽으로 이동

목표:

최대한 오래 생존하며 점수를 높이는 것이 목표입니다.

반사신경과 빠른 판단력이 필요합니다.
